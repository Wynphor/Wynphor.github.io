软件浮生录

发表于2025-11-17|更新于2026-02-14|朝花夕拾浮生录
|总字数:812|阅读时长:2分钟|浏览量:

引言

小时候我的互联网活动范围一直在社交软件和网站上，登录便捷，不留痕迹。拥有第一台手机和电脑以后，便喜欢尝试一些大众小众，用得着用不着的软件，就如我对世间事物都保持的好奇与热爱。

此篇仅记录一些相对普通用户较为小众的软件，此处只是记录一下并做主观感受, 不存在利害关系。

浮生录

编程开发

Microsoft VS Code

微软大战代码（bushi）

开源免费，自由度高，兼容性强，但对新手不太友好，建议出门右转IntelliJ全家桶

IntelliJ 系列

编程开发全家桶，总有一款适合你（）

它家常用的就CLion，IDEA，PyCharm这几个，好处是你不用花时间熟悉编程环境，坏处是运行时内存占用率贼高。

学生党可以嫖一下破解版

微信开发者工具

打比赛被赶鸭子上架学的，高易操作性势必会带来高bug复发率，企鹅多年的屎山代码依旧稳定发挥，联网，实名，备案，各种限制那么严格，也就学生和市场主流资本会进来掺和一脚了（）

EasySpider

低代码爬虫编写工具，可以用来抢课

WebRPA

一个大学生编写的小众开源项目，比EasySpider更便捷，功能更全面，可玩性很高

AI编程

国产大D的崛起，ai突然爆火，仿佛什么都可以用ai。Trae，Cursor……

对计算机领域来说，ai无非就帮你省去了最底层的代码编写。解决问题的逻辑，代码的总体架构设计，安全性兼容性问题，这些都需要人来解决，所以还是好好学习吧

Navicat Premium 16

GUI做的相对不错的数据库管理类软件

安全

VMware Workstation Pro

安装花了好几天，期间还把网卡驱动作没了，电脑才用半年被重装系统🙂，装好用了一段时间发现可以用Docker🤐

SandBoxie-Plus

体积占用小的沙箱软件

CryptoMessager_GUI

文本加密工具，线上聊天挺好用的

VeraCrypt

数据加密工具

vnt_app

可异地搭建虚拟局域网

魔法

Clash Verge

网络代理工具

Telegram

聊天软件

Tor

网速贼慢，没有可靠网桥，现在基本没啥用

实用小工具

Hexo Pro

现代化 Hexo 博客后台管理系统

Memos

一个以隐私为先、自托管的知识库，用于个人笔记、团队维基和知识管理

Typora

非常好用的Markdown编辑器

Excalidraw

开源好用的画布，应用、网页版都有，想临时写写画画可以用这个

Everything

文件检索工具

Snipaste

非常便利好用的屏幕贴图工具，先天打工人牛马圣体（bushi）

Geek

体量小的强力卸载工具

桌搭

Wallpaper Engine

动态壁纸引擎

BongoCat

超级可爱的键盘猫猫！

一些奇奇怪怪不着边际的好玩软件

genact

无意义命令行生成器，摸鱼装佬神器（）

EveryonePiano

键盘什么时候也变成乐器了……

结语

借物表

IntelliJ 系列全家桶一键激活脚本 https://ckey.run/
